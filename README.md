# Voice-Assistant-System
JARVIS Voice Assisant System is a Python-based voice assistant that can interact with the user through speech recognisation, perform tasks like opening applicaions, searching on Google or Wikipedia, playing music randomly, telling jokes, and having small talk.

This project uses speech recognisation and text to speech (TTS) to provide a hands-free assistant experience similar to  Iron Man's JARVIS.


## FEATURES
 - Greet the user according to the time of day (morning, afternoon, evening)
 - Recognise voice commands using  Google Speech Recognition
 - Speak responses using pyttsx3
 - Time & Date announcements
 - Wikipedia search with spoken summary
 - Open websites like Google, Facebook, Youtube
 - Play random  music from a  specified folder
 - Open system application: Calculator, Notepad, CMD
 - Open Calendar(Google Calendar  via browser)
 - Tell jokes and respond to basic small talk
 - Exit gracefully with a voice  command

## REQUIREMENTS
 - Python 3.11 or  higher

## How to run?
 1. Create a virtual environment:
    ```bash
    conda create -n  jarvis python=3.11 -y
    ```
 2. Activate virtual environment:
    ```bash
    conda activate jarvis
    ```

 3. Install required packages:
    ```bash
    pip install -r requirements.txt
    ```

 4. Run the JARVIS script:
    ```bash
    python jarvish.py
    ```
